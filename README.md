# Learn GenAI with JavaScript
This Repo contains all the Projects I'll build while learning Generative AI.

## Setup:
- Install NodeJS (This Repo is build on NodeJS 24)
- Clone the repo in your machine
```
git clone https://github.com/armughal189/learn-genai-js.git
```
- Jump into the directory
```
cd learn-genai-js
```
- Install all the dependentcies
```
npm install
```
Note that This Repo is following Bun Package Manager.
- Obtain A Gemini API Key from AI Studio and Copy it
- Create a File named .env
- Paste your key here so The file should look like this:
```
GEMINI_API_KEY=your_pasted_key_here_after_the_equal_sign_and_without_this_text
```
- Running a Project
```
node 1_basic_ai_chatbot\index.js
```
- Tata!
