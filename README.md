# Learn GenAI with JavaScript

This repository contains all the projects I build while learning **Generative AI** using JavaScript and Node.js.

---

## üöÄ Setup

1. **Install Node.js**  
   This repository is built with **Node.js v24**.

2. **Clone the repository**  
   ```bash
   git clone https://github.com/armughal189/learn-genai-js.git
````

3. **Navigate into the project directory**

   ```bash
   cd learn-genai-js
   ```

4. **Install dependencies**

   ```bash
   npm install
   ```

   ‚ö†Ô∏è Note: This repository is also compatible with **Bun** package manager.

5. **Obtain a Gemini API Key**

   * Get your key from **AI Studio**.
   * Create a file named `.env` in the root directory.
   * Paste your key in the following format:

     ```env
     GEMINI_API_KEY=your_api_key_here
     ```
